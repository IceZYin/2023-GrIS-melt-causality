# 2023-GrIS-melt-causality

This repository contains the analyzing and plotting scripts, processed data, and resulting figures for the paper Yin et al. (2026) "Identifying Energy Balance Drivers of Greenland Ice Sheet Surface Melt Using Causal Discovery" published on *Geophysical Research Letters* [https://doi.org/10.1029/2025GL119928](https://doi.org/10.1029/2025GL119928)

---

## Repository Structure

| Directory | Description |
|------------|------------|
| script/ | Jupyter notebooks used for data analysis and figure generation |
| fig/ | Main and supplementary figures generated by the analysis |

## Data

The analyses use output from:

- CESM2 Large Ensemble (LENS2)
- RACMO2.3p2 regional climate model
- MARv3.14.0 regional climate model

Due to file size limitations, raw model output is not included in this repository. The repository contains processed datasets required to reproduce the analyses and figures. Users interested in the original model output should obtain the data from the corresponding model archives.

---

## PCMCI+

The PCMCI+ analyses in this repository are performed using the python package Tigramite and freely available at [https://github.com/jakobrunge/tigramite](https://github.com/jakobrunge/tigramite). Several analysis scripts were adapted from the Tigramite tutorial example workflows and modified for the specific applications presented in this study.

---

## Notes

- Some scripts contain local file paths that may need modification before execution.
- Figure notebooks may generate multiple figures and supplementary figures.
- Computationally intensive preprocessing workflows performed on HPC systems are not included.
